## Building

Original build was conducted on OSX 11.2.1 with

```bash
node --version
v14.7.0
```

```bash
npm --version
6.14.7
```

1. Install dependencies

  ```bash
  npm install
  ```

2. Build the extension

  ```bash
  npm run build
  # or
  make build
  ```

3. package the extension

```bash
make package
```
